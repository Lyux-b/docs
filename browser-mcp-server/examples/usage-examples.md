## Поради по використанню

1. **Завжди чекайте завантаження елементів** перед взаємодією з ними
2. **Використовуйте специфічні селектори** для надійності
3. **Додавайте затримки** для складних операцій
4. **Перевіряйте наявність елементів** перед кліками
5. **Використовуйте try-catch** для обробки помилок

## Типові селектори

- `input[name="q"]` - поле пошуку Google
- `button[type="submit"]` - кнопка відправки форми
- `.class-name` - елемент з класом
- `#id-name` - елемент з ID
- `a[href*="example"]` - посилання з певним URL
- `input[type="text"]` - текстове поле
- `select` - випадаючий список
